# ValNet Wireless — Campaña Upsell 40/40

Herramienta interna (SPA de una sola página) para gestionar la campaña de llamadas que sube a los clientes del plan 20/20 ($999) al plan 40/40, con promoción de 3 meses al mismo precio.

**Versión en vivo:** https://lucifersamcat.github.io/upsellvalnet/

## Uso
Abre `index.html` en el navegador (doble clic) o usa el enlace de arriba. No requiere instalación ni servidor; el progreso se guarda en el navegador del agente.

## Módulos
- Lista de clientes con filtros, búsqueda y orden por antigüedad
- Panel de llamada con guión paso a paso y manejo de objeciones
- Dashboard de campaña con ingreso adicional proyectado
- Exportar resultados a CSV

Datos de clientes de ejemplo (ficticios). Edita la función seedData() en index.html o usa el botón Agregar para cargar los reales.
